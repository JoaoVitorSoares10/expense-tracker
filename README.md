# Expense Tracker 

## Descrição

O Expense Tracker é um sistema de gerenciamento de finanças pessoais. Nele é possivel ter controle do seus gastos pessoais, adicionando entradas e saídas. Esse projeto foi desenvolvido durante o Desafio 5em5, promovido pelo canal Bonieky Lacerda.
